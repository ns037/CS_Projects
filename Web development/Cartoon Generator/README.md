# Cartoon Generator
## Collaborators : Nishant Shrestha and Saurav Pandey

This project was completed as a capstone assignment to the Udemy Course - The Complete Web Developer in 2023: Zero to Mastery (https://www.udemy.com/course/the-complete-web-developer-zero-to-mastery/)


### How To Use

1. Simply install the required dependencies by running the following line on the command line:  
   pip install -r requirements.txt

2. Run the 'app.py' file by running the following command on the command line:
   python app.py
