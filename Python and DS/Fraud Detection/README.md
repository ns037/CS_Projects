# CSCI349 - Credit Card Fraud Detection Prediction
collaborated with Tung Tran.

In this project, we aimi to analyze different classification models and create a well-performing model that predicts Credit Card Fraud.

### How To Use
Download the dataset from the link provided in the python notebook - credit_card_fraud.ipynb. Simply, run all the cells and observe our analysis and model performance. Please note, only the best models are included in these python notebook. To look at other preliminary tests and hyperparameter tuning results, check out Appendix1 and Appendix2.
